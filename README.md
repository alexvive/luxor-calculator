# 🧮 Luxor Calculator

A sleek, premium-themed web calculator with a gold-accented dark interface. Built entirely with HTML, CSS, and vanilla JavaScript — no frameworks, no build steps, just pure frontend elegance.

## ✨ Features
- 💎 **Premium "Aurum" UI** – dark theme with glowing gold accents
- 🔢 **Core Operations** – `+`, `-`, `×`, `÷`, `%`
- ⌨️ **Full Keyboard Support** – numbers, operators, `Enter`, `Backspace`, `Escape`
- 📱 **Responsive Layout** – centered, mobile-friendly design
-  **Zero Dependencies** – runs instantly in any modern browser
- 🧹 **Smart Controls** – clear, backspace, and error handling with auto-reset

## 🛠️ Tech Stack
- **HTML5** – semantic structure & input handling
- **CSS3** – Flexbox/Grid, gradients, shadows, hover/active states
- **Vanilla JavaScript (ES6+)** – DOM manipulation, event listeners, safe expression evaluation
